# Changelog — HC Dinámica

## v1.0.0 — Abril 2026

Primer lanzamiento público del sistema.

**Incluye:**
- Plantilla de historia clínica del paciente (10 secciones)
- Plantilla de nota de sesión (6 secciones)
- Plantilla de primera entrevista / admisión
- Prompt maestro v4.0 para actualización de HC con LLM
- Guía de uso completa
- Ejemplos ficticios de HC y nota de sesión
- Nomenclatura con iniciales para anonimización

**Principios de diseño:**
- La HC es acumulativa: nunca se borra, solo se agrega con fecha
- Los archivos usan iniciales, no nombres completos
- El LLM es asistente de registro, no co-terapeuta
- El terapeuta tiene siempre la última palabra
